Domain Integer / Long

Domain Integer / Long

Note: Value type DomainLong is not described here as it works the same as DomainInteger, except for the larger number range.

Integer Value Types extend DomainInteger, which is a wrapper arround a Integer instance. The constructor requires a meta-object MInteger to specify lower- and upper-limit.

The integer value-type-class must implement at least one method; createInstance(). This method allows the return of typed objects after operations as ‘append’.

For example, the declaration of a new PortNumber value type with a range of [0,65535].

class PortNumber extends DomainInteger<PortNumber>{
	
	public PortNumber(final Integer value) {
		super(MInteger.create(0, 65535), value);
	}

	@Override
	protected PortNumber createInstance(final Integer value) {
		return new PortNumber(value);
	}
}

usage:

final PortNumber d1 = new PortNumber(80);
final PortNumber d2 = new PortNumber(8000);
final PortNumber d3 = d1.add(d2);

System.out.println("d1 : " +d1);
System.out.println("d2 : " +d2);
System.out.println("d3 : " +d3);
System.out.println("d1.isNegative() : " +d1.isNegative());
System.out.println("d1.isZero()     : " +d1.isZero());

result:

d1 : 80
d2 : 8000
d3 : 8080
d1.isNegative() : false
d1.isZero()     : false

Note that an exception is thrown if the value is not within the declared range:

new PortNumber(-1);

result:

DomainValueException: Value -1 exceeds minimum value 0.

calculations:

final PortNumber n1 = new PortNumber(1);
final PortNumber n2 = new PortNumber(15);
final PortNumber n3 = new PortNumber(4);
System.out.println("n1 : " + n1);
System.out.println("n2 : " + n2);
System.out.println("n3 : " + n3);

System.out.println("n2.divide(n3)  : "+ n2.divide(n3));
System.out.println("n2.multiply(n3): "+ n2.multiply(n3));
System.out.println("n1.isOne()     : "+ n1.isOne());

result:

n1 : 1
n2 : 15
n3 : 4
n2.divide(n3)  : 3
n2.multiply(n3): 60
n1.isOne()     : true

Notice that the results of divide() is truncated to an integer value.
